# Ghost Docs

Official documentation for [Ghost](https://github.com/tryghost/ghost) - located at [https://docs.ghost.org](https://docs.ghost.org)

### Development

Install Mintlify CLI to preview docs changes locally:

```
pnpm i -g mintlify
```

Start dev server with:

```
mintlify dev
```
